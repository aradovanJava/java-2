Administracija

Ova putanja (/admin/*) zaštićena je u web.xml (security-constraint + BASIC prijava). Pristup imaju samo korisnici s rolom admin iz tomcat-users.xml.

  • Prijavljeni korisnik: ana
  • Ima rolu admin: da
  • Broj korisnika u registru: 0

Podatke o prijavljenom korisniku daje kontejner: request.getRemoteUser() i request.isUserInRole("admin").
